*Tech Assistant Chatbot* 

An offline AI-powered chatbot built with Python, Streamlit, Ollama, and TinyLlama for answering technology-related questions in programming, AI, networking, and computer science.

*Key Features:*
- *Fully offline*: Runs locally via Ollama + TinyLlama, no internet required
- *Interactive UI*: Streamlit provides a clean, real-time web interface
- *Lightweight*: Python manages chatbot logic and model communication

*Purpose:* Demonstrates local LLM deployment, offline AI inference, and web-based chatbot development. Ideal for internship projects, mini projects, and AI learning.
